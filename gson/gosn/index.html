Gson简单使用

发表于2021-10-29|Gson
|字数总计:569|阅读时长:2分钟|阅读量:

Gson

一、Gson简介

Java对象和Json之间的互转，一般用的比较多的类库是Jackson、Gson和FastJson。Gson比较稳定，质量高无BUG，下面记录一下Gson的学习使用。

基本概念：

Serialization：序列化，使Java对象到Json字符串的过程。

Deserialization：反序列化，字符串转换成Java对象。

Gradle引入依赖方式：

在build.gradle中的dependencies加入以下配置

compile 'com.google.code.gson:gson:2.8.5'

Maven引入依赖方式：

<!-- https://mvnrepository.com/artifact/com.google.code.gson/gson -->
<dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.8.5</version>
</dependency>

二、Gson简单使用

2.1 Gson的创建方式一

//直接new Gson();
Gson gson = new Gson();
//Gson的两个基本方法
//gson.toJson();     将bean对象转换为json
//gson.fromJson();   将json转换为bean对象

//创建User对象进行测试
User user1 = new User("张三",20);
User user2 = new User("李四",32);
User user3 = new User("王五",19);
List<User> userList = new ArrayList<>();
userList.add(user1);
userList.add(user2);
userList.add(user3);

//toJson 将bean对象转换为json
String jsonStr = gson.toJson(user1);           //jsonStr = {"userName":"张三","age":20}

//fromJson 将json转换为bean对象
User user = gson.fromJson(jsonStr, User.class);//user = User{userName='张三', age=20}

//序列化list
String jsonList = gson.toJson(userList);
/** jsonList = [
	  {"userName":"张三","age":20},
      {"userName":"李四","age":32},
      {"userName":"王五","age":19}
   ]
*/

//反序列化list
Type type = new TypeToken<List<User>>() {}.getType();
List<User> userList1 = gson.fromJson(jsonList, type);
/** userList1 = [
	   User{userName='张三', age=20},
	   User{userName='李四', age=32}, 
	   User{userName='王五', age=19}
   ]
*/

2.2 Gson的创建方式二

使用GsonBuilder

在方式一中，使用new Gson()会创建一个带有配置选项的Gson实例，如果不想使用默认配置，那么就可以使用GsonBuilder。

Gson gson = new GsonBuilder().serializeNulls().create();

首先创建GsonBuilder,然后调用GsonBuilder提供的各种配置方法进行配置，最后调用GsonBuilder的create方法，将基于当前的配置创建一个Gson实例。

GsonBuilder的一些配置

Gson gson = new GsonBuilder()
    .serializeNulls()//当字段值为空或null时，依然对该字段进行转换
    .excludeFieldsWithoutExposeAnnotation()//不对没有用@Expose注解的属性进行操作
    .setDateFormat("yyyy-MM-dd HH:mm:ss:SSS") //时间转化为特定格式
    .setPrettyPrinting() //对结果进行格式化，增加换行
    .disableHtmlEscaping() //防止特殊字符出现乱码
    .create(); //创建gson实例
