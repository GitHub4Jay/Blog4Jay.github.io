Groovy

一、Groovy 语言简单学习

Groovy 和java有很大程度上相似，学习成本低，只是为了更好使用gradle 简单学习Groovy

在idea中打开groovy console 点击 tool->groovy console 打开

ctrl + enter 运行代码

通过def关键字定义变量和方法

1.字符串的表示

单引号和双引号都可以定义一个字符串，区别是单引号中的所有表达式都是常量字符串，而双引号可以对字符串里的表达式做运算。一个美元符号加一对花括号，花括号里放表达式，例如：{a+b}，只有一个变量时可以省略花括号，例如：name

// 这是注释
println("test1")
// 这是println的简写方式
println "test2"

// 定义变量
def i = 15
println(i)

def s = "HelloWorld"
println(s)

2.集合

// 集合定义
def list = ['hello','world']

// 添加元素
list.add()   //add方法可以用<<替代
list << 'hi'

// 取出元素
println(list.get(0))     //hello
println(list.get(2))     //hi

// List定义

3.Map

// map定义
def map = ['key':'value']
println(map.get('key'))

// 添加
map.test = 'add'
println(map.get('test'))
